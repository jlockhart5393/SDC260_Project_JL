# Web Development Practicum Portfolio Website

This project is a portfolio-style website created as part of a web development practicum course. The purpose of the website is to demonstrate foundational web development skills using semantic HTML5, CSS3 styling, responsive design techniques, and deployment through GitHub Pages.

## Introduction

This project is a personal portfolio website developed throughout multiple weeks of a practicum course. Each week focused on a different aspect of modern web development, including page structure, styling, interactivity, responsive design, and deployment.

The website was created to practice using semantic HTML5 elements for accessibility and readability, apply CSS3 for layout and visual design, and implement responsive techniques to ensure the site works across various screen sizes. It also introduces version control using GitHub and publishing a live website using GitHub Pages.

The final result is a functional, responsive website that serves as a foundation for showcasing projects and can be expanded in the future to include additional coursework or personal projects.

## Features

- Semantic HTML5 structure for improved accessibility and organization
- Responsive layout that adapts to desktop and mobile screen sizes
- Styled navigation menu with hover effects
- Project section featuring sample cybersecurity dashboards with images
- Interactive elements such as hover transitions and zoom effects
- Project summary table using `<thead>`, `<tbody>`, `<tfoot>`, and `<colgroup>`
- Contact form with multiple input fields
- Deployed live using GitHub Pages

## Technologies Used

- HTML5
- CSS3
- GitHub (version control)
- GitHub Pages (deployment)
- Visual Studio Code (development environment)

## Installation

To set up this project locally, follow these steps:

1. Clone or download the repository from GitHub.
2. Navigate to the project directory on your local machine.
3. Open the `index.html` file in a web browser.
